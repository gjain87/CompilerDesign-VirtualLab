AIM -

To check whether the input string can be classified as comment or not.

Theory -

In computer programming, a comment is a programmer-readable explanation or annotation in the source code of a computer program. They are added with the purpose of making the source code easier for humans to understand, and are generally ignored by compilers and interpreters.

Procedure -

Different Programming Languages have different format for comment. But generally most of the programming languages consider the following:

  • Single Line Comment - Starts with  //
  • Mutli Line Commment - Starts with  /* and ends with */

Therefore we need to check the string for // as starting symbol or /* as starting and */ as ending symbol to classify it as a comment.
